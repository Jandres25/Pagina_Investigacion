# Página de Investigación Web

Sitio web informativo desarrollado como trabajo práctico, que investiga y documenta tecnologías fundamentales del ecosistema web.

## Contenido

El sitio está compuesto por las siguientes páginas:

| Página                     | Descripción                                              |
| -------------------------- | -------------------------------------------------------- |
| **Inicio**                 | Portada principal del trabajo práctico                   |
| **JavaScript**             | Qué es JS, para qué sirve y su relación con ECMAScript   |
| **jQuery**                 | Biblioteca JS, su propósito y popularidad                |
| **Navegadores Web**        | Firefox y Chromium: historia, características y motores  |
| **Motores de navegadores** | Qué es un motor de renderizado y lista de los más usados |

## Tecnologías utilizadas

- HTML (XHTML 1.0 Transitional)
- CSS personalizado
- [Bootstrap 4.6](https://getbootstrap.com/docs/4.6/)

## Estructura del proyecto

```
Pagina_Investigacion/
├── html/
│   ├── index.html          # Página de inicio

│   ├── jquery.html         # Página sobre jQuery
│   ├── nav_web.html        # Página sobre Firefox y Chromium
│   └── motor_web.html      # Página sobre motores de renderizado
├── css/
│   └── style.css           # Estilos personalizados
├── img/
│   ├── diseño_web.jpg
│   ├── img_javascript.jpg
│   ├── img_jquery.jpg
│   ├── img_firefox.jpg
│   ├── img_chromium.jpg
│   └── img_motor.jpg
└── README.md
```

## Cómo ejecutarlo

Al ser un sitio estático, basta con abrir `html/index.html` directamente en el navegador. No requiere servidor ni instalación adicional.
